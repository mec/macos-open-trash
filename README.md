# Open Trash keyboard shortcut for the Finder

I want to be able to hit `command-shift-t` to open the Trash in the
Finder.

## Install

- Copy the `Open Trash.workflow` to `~/Library/Services`
- Open `System Preferences`
- Open `Keyboard`
- Open `Shortcuts` tab
- Open `App Shortcuts`
- Click `+`
- Select application `Finder`
- Menu title `Open Trash`
- Keyboard shortcut `command-shift-t`

Note: `command-shift-t` is used to show/hide the tab bar in the Finder
by default.
